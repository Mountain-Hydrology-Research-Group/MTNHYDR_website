Key variables for use in Patching code
=======================================

AT=air temperature (Celsius)
AT_maxT=daily max temp
AT_minT=daily min temp
AT_meanT=daily mean temp
AT_rangeT=daily temp range

lat=latitude
lon=longtitude


elev=elevation (meters)

TIME=time matrix constructed using time_builder

Available Record: 2008 water year (1.5 hour timestep)
