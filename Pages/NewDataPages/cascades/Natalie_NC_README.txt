NCibuttons.dat

This data has been trimmed so that it does not include times recorded by ibuttons when not deployed.  This file was constructed to include only days when data was recorded on all ibuttons in the data set

formatted as follows:
[year;month;day(in PST);hour(PST); interpolated temperature for sensors 1:30, 33:45]

-999 was used for bad data in sensors 1, 11, and 35

NaNs may be present on the first and last day of each file, when there was not data for those hours

[sensor number, latitude N, longitude W, elevation]
