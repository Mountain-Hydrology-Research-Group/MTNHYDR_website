Rainier_ibuttons.dat

This data has NOT been trimmed, as date of deployment was missing for 1/4 sensors.  This file was constructed to include only days when data was recorded on all ibuttons in the data set

formatted as follows:
[year;month;day(in PST);hour(PST); interpolated temperature for sensors 1:4]

NaNs may be present on the first and last day of each file, when there was not data for those hours
